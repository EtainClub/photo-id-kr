# 여권사진 준비

한국 여권 디지털 사진을 스마트폰에서 촬영·점검·저장하는 모바일 우선 웹앱입니다. 모든 화면 문구는 한국어로 제공하며 사진은 서버에 업로드하지 않고 브라우저 안에서 처리합니다.

## 실행

```sh
npm start
```

`http://localhost:4173`에서 열 수 있습니다. 카메라 기능은 배포 환경에서 HTTPS가 필요합니다.

## Firebase Hosting 배포

```sh
npm run build:firebase
firebase serve --only hosting
npm run deploy:firebase
```

`firebase.json`은 `dist/client`의 정적 파일만 배포합니다. `npm run deploy:firebase`를 실행하면 Firebase의 배포 전 훅이 `build:firebase`를 다시 실행하므로, 실제 배포에는 최신 CSS·이미지·모델·WASM 파일이 포함됩니다.

## 주요 기능

- 촬영 화면의 35:45 프레임, 얼굴 타원, 눈높이와 실시간 밝기·얼굴 크기·중앙·수평 안내
- 앨범 사진 불러오기, 로컬 얼굴 감지를 통한 자동 위치 맞춤
- 손가락 드래그·두 손가락 확대를 이용한 비파괴 자르기
- 해상도, 흰 배경, 조명, 선명도, 얼굴 중앙, 머리 크기, 고개 수평 자동 점검
- 표정·영유아 예외, 눈·안경, 장신구·종교 의상, 다른 사람·사물, 최근 촬영 여부 직접 확인
- 413×531 픽셀, 300dpi 표기, 500KB 이하 JPG 생성
- 외교부 온라인 여권 사진 검증으로 바로 이동
- 설치형 웹앱과 오프라인 재사용 지원

## 원칙

외교부 규정상 배경 삭제, 얼굴 보정, 필터, AI 합성 사진은 허용되지 않을 수 있으므로 앱은 원본 비율을 유지한 자르기와 위치 조정만 수행합니다. 자동 점검은 발급을 보장하지 않으며 최종 제출 전 외교부 검증을 권장합니다.
